Trout Stream Special Regulations

This page last updated: 2009-01-02 08:24:34.0
Metadata created using Minnesota Geographic Metadata Guidelines

Metadata Summary

Originator Minnesota DNR - Division of Fish & Wildlife - Fisheries Unit
Abstract This layer shows Minnesota trout streams that have a special regulation as described in the 2006 Minnesota Fishing Regulations. Road crossings were determined using the 1991 Air Photos (1 - metre DOQs). Check the Minnesota Fishing Regulations for detailed information. See http://www.dnr.state.mn.us/rlp/index.html for legal descriptions and restrictions associated with designated trout waters.
Use Tips This data layer is a subset of the DNR 24K River and Streams layer, a statewide data set originally derived from USGS seven and one-half minute quadrangle maps. Users are encouraged to use the full resolution DNR 24K Rivers and Streams layer as a base layer to visualize all streams and fully understand hydrological connectivity, watershed position and spatial relationships to designated trout stream resources.

Data Elements for Attribute Table STRM_REGSLN3.DBF:
